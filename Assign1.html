Form tags as follows:

"form" is the parent tag for an HTML form.
"action" tag used for the URL listed here is where the form data will be submitted once user fills it.
"method" tag - It specifies which HTTP method (POST or GET) would be used to submit the form.
"enctype" tag - Only for POST method, this dictates the data encoding scheme to be used when form is submitted.
"autocomplete" tag - Determines if the form has auto-complete enabled.
"novalidate" tag - Determines whether the form should be validated before submission.
"accept-charsets" tag - Determines character encodings when form is submitted.
"target" tag - After submission, the form response is displayed wherever this refers to, usually has the following values: _blank, _self, _parent, _top
"fieldset" tag - Identifies the group of all fields on the form.
"legend" tag - This operates as a caption for the 'fieldset'element. "input" tag is used to take input from the user.Input type is determined by a number of attributes.

Input type attributes

"type" tag - Determines which type of input (text, dates,password) is requested from the user.
"name" tag - Specifies the name of the input field.
"value" tag - Specifies the value contained currently in the input field.
"size" tag - Determines the input element width (number of characters).
"maxlength tag - Specifies the most input field characters allowed.
"required" tag - Makes an input field compulsory to be filled by the user. The form cannot be submitted if a required field is left empty.
"width" tag - Determines the width of the input element,in pixel values.
"height" tag - Determines the height of the input element,in pixel values.
"placeholder" tag - Can be used to give hints to the user about the nature of the requested data.
"pattern" tag - Specifies a regular expression, which can be used to look for patterns in the user’s text.
"min" tag - The minimum value allowed for an input element.
The maximum value allowed for an input element.
"autofocus" tag - Forces focus on the input element when webpage loads completely.
"disabled" tag - Disables the input element. User can no longer enter data.

Select Attributes

"name" tag - The name of a particular list of options.
"size" tag - Total number of options given to the user.
"multiple" tag - States whether the user can choose multiple options from the list.
"required" tag - Specifies whether choosing an option/s is necessary for form submission.
"autofocus" tag - Specifies that a drop-down list automatically comes into focus after a page loads.

Option Attributes

"value" tag - The text visible to the user for any given option.
"selected" tag - Determines which option is selected by default when the form loads.
